# d4jclone
Python clone of [defects4j](https://github.com/rjust/defects4j) with defects from the [smartSHARK](https://smartshark.github.io/) database.
## install
```
python3 setup.py install --user
```
Add to path
```
export PATH=$PATH:/home/"USER"/.local/bin
```
